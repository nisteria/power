# Power — Austrian Energy Orchestrator

Power ist ein österreichisches Energy-Startup ("Made in Austria") für intelligente Steuerung von Stromverbrauchern und Speichern auf Basis dynamischer Stromtarife.

## Vision
Haushalte und KMU laden, speichern und verbrauchen Strom automatisch dann, wenn es wirtschaftlich und netzdienlich sinnvoll ist.

## Produktkern
- Energy Hub (lokales Gateway + Steuerlogik)
- Web-Onboarding + Mobile App
- Tarif-Engine (preis- und regelbasiertes Laden/Entladen)
- Geräte-Orchestrierung (Batterie, Wallbox, Boiler, Wärmepumpe)
- Installateur-Portal (Inbetriebnahme/Service)

## Markteintritt (AT)
1. **Phase 1:** Software + Integrationen, kein eigener Stromlieferant
2. **Phase 2:** Assembly/Hardware in Österreich + Serviceverträge
3. **Phase 3:** White-Label/Partner-Tarife mit Lieferanten
4. **Phase 4:** Optionaler Lieferantenstatus nach Skalierung
