# BOOTSTRAP.md — Power Projekt-Start

Wenn dieses Repo frisch geöffnet wird:

1. Lies in Reihenfolge:
   - `SOUL.md`
   - `USER.md`
   - `PHASE_PLAN.md`
   - `PROJECT_TODO.md`
   - `PROJECT_STATUS.md`
2. Setze `Aktive Phase` in den Fokus.
3. Führe genau einen ersten konkreten Schritt aus.
4. Dokumentiere in:
   - `PROJECT_STATUS.md`
   - `POWER_AGENT_ACTIVITY.md`

## Arbeitsregel
Nur aktive Phase abarbeiten. Bei Phasenabschluss stoppen und Freigabe bei Lech holen.
